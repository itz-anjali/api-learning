# api-learning
API learning from zero to advanced with hands-on practice, real examples, and Python-based API usage.
<hr>
This repository documents my journey of <b>learning APIs </b> from scratch to advanced level.

<strong> Topics covered: </strong>
<ol>
- API fundamentals (client–server, request–response)
- HTTP methods (GET, POST, PUT, DELETE)
- JSON handling
- API testing with Postman
- Python-based API integration
- Authentication (API keys, basic auth)
- Error handling
- Mini projects and real-world API use cases
</ol>
Goal:
To gain practical, job-ready API knowledge and use APIs confidently in AI, automation, and real-world projects.
